第一篇文章记录Markdown的语法

 2020-03-07  2020-03-09
  Markdown
  5 分钟 读完 (大约 742 个字) 0次访问

我对于标记语言 Markdown 用的并不多，只知道一些常用的语法。第一篇文章熟悉一下环境，对Markdown的语法一点记录，以便后期查阅。

标题

# 标题文字（1个#是一级标题，2个#是二级标题，以此类推）
⚠️中间有空格

输出：
这是一级标题
这是二级标题
这是三级标题
这是四级标题
这是五级标题
这是六级标题

代码

  • 单行代码
    用反引号` 包住代码 ，即`代码块`

例：`print("hello")`

输出：print("hello")

  • 代码块
    用三个反引号` 包住代码块 且每个反引号换行

```
代码块
```
对于Hexo博客且安装highlight.js实现代码高亮 可有以下参数

``` [language] [title] [url] [link text]
代码块
```

输出：

列表

无序列表

- 内容
⚠️中间有空格, - + * 均可，代表不同种类

输出：

  • 狮子🦁️
  • 老虎🐯
  • 苹果🍎
  • 桔子🍊
  • 向日葵🌻
  • 玫瑰 🌹

有序列表

数字. 内容
⚠️中间有空格

输出：

  1. 把冰箱门打开
  2. 把🐘放进去
  3. 把冰箱门关上

字体

  • 加粗
    **包起来
  • 斜体
    *包起来
  • 斜体加粗
    ***包起来
  • 删除线
    ~~包起来

输出：
加粗
斜体
加粗斜体
加删除线

引用

>引用的内容
⚠️中间无空格，可以加多个来嵌套

输出：

引用内容

引用内容

引用内容

引用内容

分割线

三个及以上的-或*

输出：

图片

![caption](图片地址 ‘’鼠标在图片上显示的文字’’)

例：![哈莉·奎茵（Harley Quinn）](http://ent.k618.cn/ylzx/201610/W020161015180432690118.jpeg "小丑女")

输出：

超链接

[超链接名](超链接地址 “超链接title”)

例：[百度](http://baidu.com)

输出：
百度

表格

⚠️-有一个就可以 多个为了对齐
默认左对齐 两边加:居中 右边加:居右

输出：

姓名 性别 年龄
小明 男 12
小花 女 24
